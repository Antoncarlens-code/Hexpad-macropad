# README images

Drop the images the top-level README references here (Hackpad requires all four):

- `render.png` — a full render showing **all** parts (KiCad 3D viewer or CAD).
- `schematic.png` — screenshot of the KiCad schematic.
- `pcb.png` — KiCad PCB render or a photo of the real board.
- `case.png` — photo of the printed case.
